VGG图像分类

首先将下载预训练好的vgg-19模型，用scipy读取imgenet-vgg-verydeep-19.mat；

定义网络层数：5层layers，第一层和第二层先是2个卷积+relu，然后加1个maxpooling层，
第三、第四和第五层先是3个卷积+relu，在加1个maxpooling层，vgg-19需要对图片进行
预处理，把每个像素点的取值减去训练集算出来的RGB均值，各层的定义，transpose
反转函数，对应高、宽、输入通道、卷积核数目

定义模型：建立卷积、最大池化模型，预处理、读取和保存图片的操作

运行：预训练，初始化vgg-19网络，读入猫图片，通过各层提取图片特征，并用matplotlib
显示每层的特征
